# DR-FR0003 - Lista objętych posterunków

:::warning
Wymaga uzupełnienia. Brak UC dla opisanego FR.
:::

|    Wykonane     | Powiązane UC | Blokuje | Jest blokowane przez |
| :-------------: | :----------- | :------ | -------------------- |
| <FaRegSquare /> |              |         |                      |

Po objęciu dowolnego posterunku, czy całej scenerii, jest ona automatycznie dodawana do listy wszystkich objętych scenerii przez danego Dyżurnego Ruchu. Lista ta jest podstawowym narzędziem dla DR pozwalającym przełączać się pomiędzy poszczególnymi narzędziami pozwalającymi na prowadzenie ruchu na daje scenerii czy posterunku zgodnie z tym, jakie narzędzia zostały oznaczone jako dostępne.

:::info
Na różnych posterunkach dostępne mogą być różnego tpy narzędzia do prowadzenia ruchu. To, jakie narzędzie jest dostepne określane jest przez autora scenerii w porozumieniu z zespołem SSS. Dyżurując na małej stacyjce, bądź posterunku odstępowym mozna mieć dostępne tylko książki papierowe, a na wiekszych posterunkach mozna mieć systemy elektroniczne.
:::
Lista jest integralną częścią modułu "DR".
