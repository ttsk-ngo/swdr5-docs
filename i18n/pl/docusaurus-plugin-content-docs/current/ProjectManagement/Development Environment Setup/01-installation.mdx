# Instalacja

Poniższa instrukcja jest sugestią, jak należy zainstalować i skonfigurować środowisko, aby móc tworzyć dokumentację korzystając z pełnego frameworka Docusaurus. Jeżeli preferujesz inne narzędzia, nic nie stoi na przeszkodzie, abyś z nich korzystał. Instrukcja omawia tylko środowisko Microsoft Windows.

## Repozytorium

## Power Shell

Wszystkie poniższe operacje zakładają zainstalowanie aplikacji, które będą dostępne dla wszystkich użytkowników. Do tego celu wymagane są uprawnienia administratora. Jeżeli wolisz zainstalować poniższe aplikacje tylko dla bieżącego użytkownika, pomiń opcję `--scope machine` we wszystkich dalszych poleceniach.

Uruchom PowerShell z uprawnieniami Administratora

![logo](/img/ProjectManagement/DevelopmentEnvironmentSetup/RunPowershellAsAdmin.png)

### Git

winget install --id Git.Git -e --source winget --scope machine
