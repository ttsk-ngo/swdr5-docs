# Pobranie Repozytorium

## Logowanie do Github

1. Uruchom aplikację GitHub Desktop
2. Zaloguj się do swojego konta GitHub. Jezeli nie masz konta, skorzystaj z odnośnika do jego utworzenia.
![GitHub01](/img/ProjectManagement/DevelopmentEnvironmentSetup/01-GithubWelcomeScreen.png)
3. Dostosuj nazwę konta do tej, którą masz na forum TrainDriver 2. Ułatwi to identyfikację autora.
![GitHub02](/img/ProjectManagement/DevelopmentEnvironmentSetup/02-GithubUsername.png)
4. Wybierz "kolonowanie"
4.1 Podaj URL: https://github.com/ttsk-ngo/swdr5-docs.git
4.2 Po pobraniu repozytorium, utworz nową gałąź, której nazwa będzie odnosić się do zmian, które chcesz wprowadzić np: feature/FR0009 albo fix/FR0009 Ewentualnie wybierz istniejący branch w którym chcesz wprowadzać zmiany.
:::info
Więcej o pracy z git znajdziesz następnym punkcie.
:::
