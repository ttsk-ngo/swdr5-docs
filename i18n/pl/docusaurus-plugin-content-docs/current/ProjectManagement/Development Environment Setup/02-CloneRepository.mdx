# Pobranie Repozytorium

## Logowanie do Github

1. Uruchom aplikację GitHub Desktop
2. Zaloguj się do swojego konta GitHub. Jezeli nie masz konta, skorzystaj z odnośnika do jego utworzenia.
![GitHub01](/img/ProjectManagement/DevelopmentEnvironmentSetup/01-GithubWelcomeScreen.png)
3. Dostosuj nazwę konta do tej, którą masz na forum TrainDriver 2. Ułatwi to identyfikację autora.
![GitHub02](/img/ProjectManagement/DevelopmentEnvironmentSetup/02-GithubUsername.png)
4. Wybierz "Clone a repository from the Internet"
![GitHub03](/img/ProjectManagement/DevelopmentEnvironmentSetup/03-GithubCloneRepo01.png)
    - Podaj URL: https://github.com/ttsk-ngo/swdr5-docs.git
    - Wskaż miejsce na dysku, gdzie chcesz sklonować repozytorium.
![GitHub04](/img/ProjectManagement/DevelopmentEnvironmentSetup/04-GithubCloneRepo02.png)
5. Po pobraniu repozytorium wybierz opcję `To contribute...`
![GitHub02](/img/ProjectManagement/DevelopmentEnvironmentSetup/05-GithubContribute01.png)
:::info
Reporyztorium jest gotowe do pracy.
:::
